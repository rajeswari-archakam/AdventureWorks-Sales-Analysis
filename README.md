 AdventureWorks Sales Analysis

## Project Overview
This project analyzes the AdventureWorks sales dataset to generate meaningful business insights using Excel, SQL, Power BI, and Tableau. It focuses on sales performance, customer trends, regional analysis, and product performance through interactive dashboards.

## My Role
- Performed data cleaning and transformation using Excel.
- Created Pivot Tables, Pivot Charts, KPIs, and the final Excel Dashboard.
- Developed an interactive Tableau Dashboard with charts, filters, calculated fields, and visualizations.
- Validated business insights and collaborated with the team to integrate all project modules.

## Team Contribution
- *My Role:* Excel Dashboard, Tableau Dashboard, Data Cleaning & Visualization.
- *Team Members:* SQL Queries, Data Analysis, Power BI Dashboard, DAX Measures, Data Modeling.

## Tools Used
- Microsoft Excel
- SQL
- Tableau
- Power BI

## Key Business Insights
- Analyzed year-wise and month-wise sales performance.
- Identified top-performing products and customers.
- Compared sales versus production cost.
- Evaluated regional sales performance.
- Built interactive dashboards to support business decision-making.
